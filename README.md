# Lector
"Lector Rápido Flash" es una app web de un solo archivo para entrenar tu lectura. Pega tu texto o usa ejemplos integrados. Ajusta la velocidad logarítmica desde modo "Caracol" hasta un frenético nivel "Insane", elige cuántas palabras ver a la vez y concéntrate al máximo con su inmersivo modo de pantalla completa.
